You are using CicleDrop, a file transfer server application for use within the same LAN.

The app should be accessible by any device with a browser. It provides a file tree structure with a limited functionality to upload, download, and delete files and folders. One computer in your network needs to run the server, and other devices can connect to it using its local IP address followed by the port number (default is 8080). Cookies are currently used for sessions and upload history tracking.
